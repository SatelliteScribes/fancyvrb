# Rampage-Trainer-Rus
Русский перевод для трейнера Rampage, который используется для игры GTA5.

Установить файл Russian.xml по примеру пути C:\Games\Grand Theft Auto V\RampageFiles\Language

Над русификатором работают: Canek77, yefan_p

Для поддержки донатом: https://www.donationalerts.ru/r/canek77
